Excel 必備公式精選：文書處理檢定與職場攻略

📅 更新：2026 🔎 主題：文書處理 💼 適用：職場技能與檢定

在「電腦軟體應用」技術士檢定中，Excel 往往是決定及格與否的關鍵。許多考生會卡在複雜的公式巢狀結構中。其實，只要掌握核心的 5 個函數組合，就能解決 80% 的題庫問題。本文幫你精煉出最實用的公式重點，讓你不再被試算表難倒。

本篇攻略快速導覽 1. 萬能查詢：VLOOKUP 2. 智慧判斷：IF 邏輯 3. 統計神隊友：COUNTIF 與 SUMIF 4. 字串處理：LEFT、RIGHT 與 MID

一、萬能查詢：VLOOKUP 函數

這是考場出現率第一名的函數！它能幫你從大量資料表中，根據「關鍵字」抓取出對應的數值（如：根據學號抓取成績）。

公式結構： =VLOOKUP(找什麼, 在哪找, 第幾欄, 0)
考場撇步： 最後一個參數請務必填 0 (代表精確搜尋)，這在 99% 的考題中都是正確解答。

二、智慧判斷：IF 邏輯函數

IF 函數是電腦具備「判斷能力」的基礎。當你需要根據成績標記「及格」或「不及格」時，就是它上場的時候。

  • 基本用法： =IF(分數>=60, "及格", "不及格")
  • 巢狀用法： 如果有多種等級（A, B, C），則需要在 IF 裡面再套一個 IF。

三、統計神隊友：COUNTIF 與 SUMIF

這兩個函數是專門用來處理「有條件的統計」。

函數名稱 主要用途 語法範例
COUNTIF 計算個數 (有幾人及格?) =COUNTIF(範圍, ">=60")
SUMIF 加總數值 (總共發多少獎金?) =SUMIF(部門範圍, "業務部", 薪資範圍)
AVERAGE 計算平均值 =AVERAGE(範圍)

四、字串處理：LEFT、RIGHT 與 MID

常用於提取特定欄位資訊，例如從身分證字號中提取性別碼，或從學號中提取入學年份。

  • LEFT(網址, 5)：取最左邊 5 個字。
  • RIGHT(電話, 3)：取最後面 3 個字。
  • MID(身分證, 2, 1)：從第 2 位開始，取 1 個字。
⚠️ 考前注意：絕對路徑與相對路徑
在拖曳公式填充時，若某個儲存格（例如：匯率）必須固定，別忘了按下 F4 加上錢字號 (如 $A$1)，否則計算結果會全盤皆錯。
