英文不卡關：資訊類檢定高頻率專業單字表

📅 更新：115 年度最新版本 🔎 主題：資訊類檢定 / 專業英文

在準備「電腦硬體裝修」、「網路架設」或「電腦軟體應用」等資訊類技術士檢定時，許多考生最怕遇到的不是技術操作，而是充滿英文縮寫的考題與錯誤代碼（Error Message）。其實只要掌握常考的專業詞彙，在閱讀題目或除錯時就能大幅提升判斷速度，輕鬆拿分！

📌 本篇攻略快速導覽 一、硬體與系統基礎單字 二、網路與通訊必考名詞 三、軟體與資訊安全術語 四、高分背誦技巧與注意事項

一、硬體與系統基礎單字

硬體裝修與作業系統安裝是檢定的基本功，以下單字在 BIOS 設定與硬體故障排除題目中出現頻率極高。

英文簡稱 / 單字 中文翻譯 檢定常考情境說明
Motherboard (MB) 主機板 所有硬體連接的樞紐，常出現在組裝與相容性考題。
BIOS / UEFI 基本輸入輸出系統 開機順序 (Boot Sequence)、系統時間調整必經介面。
RAM (Random Access Memory) 隨機存取記憶體 影響系統暫存與運行速度，常考容量單位與插槽規格。
SSD (Solid State Drive) 固態硬碟 現今主流儲存裝置，常與傳統硬碟 (HDD) 比較讀寫速度。

二、網路與通訊必考名詞

無論是乙太網路壓線或是伺服器架設，網路概念都是各類檢定的命題核心。

英文簡稱 / 單字 中文翻譯 檢定常考情境說明
IP Address 網際網路協定位址 IPv4 與 IPv6 的格式、網段劃分 (Subnet Mask) 計算。
Router 路由器 負責跨網段資料傳輸，網路架設丙/乙級必考設備。
Protocol 通訊協定 例如 TCP/IP、HTTP、FTP 等，決定資料如何傳輸的規範。
Bandwidth 頻寬 網路傳輸速率的測量，常考單位換算（如 Mbps vs MB/s）。

三、軟體與資訊安全術語

隨著資安意識抬頭，病毒防護、權限設定及加密協定已成為學科測驗的新重點。

英文簡稱 / 單字 中文翻譯 檢定常考情境說明
Firewall 防火牆 阻擋惡意連線與封包過濾，常見於系統安全設定。
Malware 惡意軟體 統稱病毒 (Virus)、木馬 (Trojan) 等有害程式。
Encryption 加密 保護資料傳輸安全，常與解密 (Decryption) 一起考。

四、高分背誦技巧與注意事項

背誦專業英文與一般生活英文不同，掌握字根字首與情境聯想才是最快的方法。

👨‍🏫 老師說： 許多單字在檢定中會以縮寫出現，建議大家先理解全名的意思，記憶起來會事半功倍！例如看到「HTTP」，若知道它是 HyperText Transfer Protocol，就能馬上聯想到網頁與傳輸。
⚠️ 考生避坑警告：
  • 注意大小寫差異：例如 b (bit/位元) 與 B (Byte/位元組) 在計算容量或網速時差了 8 倍！
  • 不要死背拼音：請將英文單字直接與「實體設備」或「操作畫面」做連結。
